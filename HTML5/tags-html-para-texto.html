Procure sempre usar as tags pelo sentido semântico

A tag bold deixa o texto em negrito, muda o texto físico
Já a tag strong deiax o texto mais forte! Portanto , muda o estado semântico.
A tag itálico deixa tudo italico, muda o estado físico
Dá ênfase a palavra, muda o estado semântico.
Procure sempre usar as tags pelo sentido físico semântico!
